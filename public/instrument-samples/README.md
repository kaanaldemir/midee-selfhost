Instrument sample attribution
=============================

The MP3 sample sets in this directory are adapted from
nbrosowsky/tonejs-instruments:

https://github.com/nbrosowsky/tonejs-instruments

Code in that project is MIT licensed. The samples are released under
Creative Commons Attribution 3.0:

https://creativecommons.org/licenses/by/3.0/

The upstream project notes that the samples come from public-domain and
freely available sources, including VSCO 2 Community Edition, University of
Iowa Musical Instrument Samples, Karoryfer Samples, and Freesound.
